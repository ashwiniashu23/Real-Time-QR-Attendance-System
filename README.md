# Real-Time QR Attendance System

## Overview
A real-time attendance tracking system that records student attendance by scanning a QR code only when the student is within the permitted distance, ensuring secure and accurate attendance management.

## Features
- QR code-based attendance
- Real-time attendance tracking
- Distance-based verification
- Secure attendance recording
- Student attendance management

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## How It Works
1. The administrator generates a QR code.
2. Students scan the QR code.
3. The system verifies that the student is within the allowed distance.
4. Attendance is recorded automatically in the database.
